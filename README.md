# Customer Churn Prediction

## Project Overview
Developed a Machine Learning model to predict customer churn and identify factors contributing to customer retention.

## Key Features
- Performed data preprocessing and feature engineering on customer data.
- Conducted exploratory data analysis to identify churn indicators.
- Trained and evaluated machine learning models for churn prediction.
- Assessed model performance using Accuracy, Precision, Recall, F1 Score, and Confusion Matrix.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Performance Metrics
- Accuracy
- Confusion Metrics
- Classification Report
